# Glossary

| Term | Definition |
|------|-------------|
| **Blocking** | Planned actor movement on stage. |
| **Cue** | Signal for an action, light, or sound change. |
| **Projection** | Speaking or singing so the entire audience can hear clearly. |
| **Stage Right / Left** | The actor's right or left while facing the audience. |
| **Tech Run** | Full rehearsal with lighting, sound, and props. |
| **Servant Leadership** | Leading by serving others, following Christ's example. |
