# Checklists

## Rehearsal Readiness
- [ ] Script and music printed
- [ ] Stage cleared and marked
- [ ] Sound check completed
- [ ] Prayer before rehearsal

## Tech Run
- [ ] All cues tested (lighting, sound, props)
- [ ] Communication headsets working
- [ ] Backup media verified
- [ ] Safety walk-through complete

## Day-of-Performance
- [ ] Call times confirmed
- [ ] Costumes and props organized
- [ ] Team devotion and prayer
- [ ] Emergency plan reviewed
